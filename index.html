Position

Ideas principales


Position

position: static:
Es el valor predeterminado para todos los elementos. Los elementos se colocan de acuerdo con el flujo normal del documento, es decir, en el orden en que aparecen en el HTML. No se puede manipular con las propiedades top, left, bottom o right.
position: relative:
El elemento se posiciona en relación con su ubicación normal en el flujo del documento. Se pueden ajustar su posición usando las propiedades top, left, bottom o right. El espacio original que ocupa el elemento en el flujo del documento no se modifica, aunque su posición visual cambie.
position: absolute:
El elemento se retira del flujo normal del documento y se posiciona en relación con su contenedor más cercano que tenga una propiedad position distinta de static. Se puede mover libremente en la página usando top, left, bottom, y right. Otros elementos no ocupan su espacio en el flujo del documento.
position: fixed:
El elemento se posiciona en relación con la ventana del navegador (viewport). No se mueve cuando se hace scroll en la página. Es útil para crear menús o barras de navegación fijas en la parte superior o inferior de la pantalla.
position: sticky:
Combina características de relative y fixed. El elemento actúa de forma relativa hasta que el usuario hace scroll hasta un punto definido, momento en el que se "adhiere" al viewport como si fuera fixed. Requiere definir las propiedades de top, left, bottom o right para especificar cuándo el elemento debería fijarse.



Apuntes


Position

Static:
Los elementos con position: static siguen el flujo normal del documento. En pocas palabras esta donde lo pusiste.

Relative:
Nos permite mover la posicion de los objetos, pero como quien dice se respeta el lugar original de este objeto.

Absolute:
Nos permite mover del objeto donde queramos pero al quitarlo los demas objetos actuan como si ya no estuviera.

sticky:
Es una mezcla de relative y fixed, Basicamente el objeto al hacer un scroll no va desaparecer de la pantalla.

Resumen

Estas propiedades nos sirven para controlar cómo quieres que se muestre la información y el estilo que quieras darle a tu página web. Con ellas puedes ajustar la posición de los elementos para crear diseños más organizados y atractivos

Display

Ideas principales


Display

Block:
Hace que el elemento ocupe todo el ancho disponible, empujando a otros elementos hacia abajo.
Inline:
Hace que el elemento ocupe el espacio que necesita, y permite que mas elementos esten en la misma linea.
Inline-Block:
Es una mezcla entre block y inline, se comporta como un inline pero te respeta el ancho y alto que le asignas.
None:
Hace que el elemento no se muestre, o se puede poner un hover para que se ponga el cursor encima desaparece.
Flex:
Permite organizar los elementos hijos, en filas o columnas.
Grid:
Convierte el contenedor en una cuadricula.



Apuntes

Animacion

Ideas principales


Animacion

Animation:
En este se hace un cuadro que al abrir la pagina se va haciendo grande, dependiendo de como ordenes el color y las dimensiones.
Animation funtion:
Linear:
Mueve el objeto de forma linean apartir de las propiedades del objeto.
Ease:
Crea una velocidad constante y mas suave.
Ease in:
El elemento se posiciona en relación con la ventana del navegador (viewport). No se mueve cuando se hace scroll en la página. Es útil para crear menús o barras de navegación fijas en la parte superior o inferior de la pantalla.
Ease out:
Comienza con una aceleracion suave luego rapida y termnina en lenta.
Cubic bezier:
Permite definir la aceleración y desaceleración de una animación a través de una curva cúbica de Bézier.
Animation direction:
Normal
La animación se reproduce desde el inicio hasta el final (de 0% a 100%).
Reverse:
La animación se reproduce desde el final hasta el inicio (de 100% a 0%).
Alternate:
La animación se reproduce desde el inicio al final y luego de nuevo al inicio, alternando la dirección en cada ciclo.
Alternate-reverse:
La animación se reproduce desde el final al inicio y luego de nuevo al final, alternando la dirección en cada ciclo.



Apuntes


Animacion

None:
Este es el valor por defecto. No se aplican estilos antes o después de la animación; el elemento regresa a su estado original cuando la animación termina.
Forwards:
El elemento mantendrá el estilo del último fotograma de la animación cuando termine la ejecución. Es decir, después de que la animación ha finalizado, el elemento conservará el estilo definido en el 100% de la animación.
Backwards:
El elemento tomará el estilo del primer fotograma de la animación antes de que comience la animación. Si la animación se retrasa (debido a la propiedad animation-delay), el estilo del primer fotograma se aplicará inmediatamente.
Both:
Combina los efectos de forwards y backwards. Mantiene el estilo del último fotograma después de que la animación ha terminado y también aplica el estilo del primer fotograma antes de que comience la animación.

Resumen

Las animaciones son muy utiles para darle animaciones fluidas, diferentes y muy llamativas para el usuario.

Atributo id El atributo id es como un nombre que identifica al elemento. El valor de este atributo no puede ser duplicado. Este nombre debe ser único en todo el documento.
Elementos un selector de elementos en CSS es una regla que selecciona todos los elementos de un tipo especÍfico en el documento HTML.
Pseudoclases - :nth-xhild()se utiliza para seleccionar uno o más elementos en función de su posición dentro de su elemento padre. Es especialmente útil para aplicar estilos a elementos específicos en una lista o una tabla sin necesidad de agregar clases adicionales.
Atributo class este atributo es mas flexible y puede ser asignado a varios elementos HTML en el documento que comparten diseño similar.
